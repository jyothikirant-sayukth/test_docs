# <u>Terminology</u>
+ <b> Back :</b> The action of moving users current screen to the previous screen.
+ <b> Backup :</b> Transferring the data from one mobile to another mobile.
+ <b> Citizen : </b> A person who lives in their native panchayat. 
+ <b> Complete Survey :</b> To close one village/panchayat in which survey is completed.
+ <b> Confirmation screen :</b> To verify the details that were given in the create screen.
+ <b> Create Screen :</b> To give the inputs and to create the new records (e.g., creating a new house).
+ <b> Dashboard :</b> The Home Screen displaying an overview or summary of app features.
+ <b> Delete :</b> To remove a record from the list screen.
+ <b> Device Compatibility : </b> It checks overall device compatibility status to ensure whether the device is supported or not.
+ <b> Done :</b> The action to exit the task when it is completed.
+ <b> Edit :</b> The action of modifying or updating existing information in the records.
+ <b> FAQ (Frequently Asked Questions) :</b> A section that answers common questions about the app.
+ <b> Finish :</b> The action of completing a record and saving it in the list screen.
+ <b> Forgot Password :</b> A feature for users to reset their password, if they forgot it.
+ <b> GIS Map :</b> A visual representation of geographic data or locations.
+ <b> Help :</b> A section providing assistance and instructions to guide users.
+ <b> Information Icon :</b> It provides extra information about the feature.
+ <b> List Screen :</b> The screen that displays a list of created records.
+ <b> Login :</b> The process of entering your credentials (username and password) to access the app.
+ <b> Logout :</b> The action of exit from the app.
+ <b> Menu :</b> The main navigation area to access different features of the app.
+ <b> Next :</b> The action of moving users Current Screen to the next screen.
+ <b> Notifications :</b> Alerts sent to the app to inform users about updates or any information related to app.
+ <b> OCR(Optical Character Recognition) :</b> It captures a photo of the Aadhaar Card and displays the details of the captured photo on the screen.
+ <b> Profile :</b> The section where users can view their personal details.
+ <b> Pin Current Location :</b> Pin will be marked near to the current location to pin the current location.
+ <b> Pin Landmark location :</b> Pin will be marked on landmark location.
+ <b> Pin Previous Location :</b> Pin will be marked on Previous record pin location.
+ <b> Record :</b> A record is a digital entry created after a survey, capturing detailed information about a property and its residents.
+ <b> Registration: </b> The process of creating a user account by providing necessary details to open and use the app.
+ <b> Search Bar :</b> Located at the top of the list screen, enabling you to find specific records.
+ <b> Settings :</b>  The setting that allows users to select their preferred language for the application interface.
+ <b> Statistics :</b> It displays the count of the records in list screen.
+ <b> Survey :</b> The process of collecting data about specific property.
+ <b> Terms & Conditions :</b> The legal agreement between the user and the app, outlining rules and responsibilities.
+ <b> Title :</b> The Heading that is visible on the top of each screen.
+ <b> Troubleshooting :</b> Troubleshooting is the process of finding and solving problems with a device or any system.
+ <b> Upload :</b> The action of sending data from Survey App to Panchayat Seva Cloud Server.
+ <b> User :</b> A user is a person who uses a product or an app.
+ <b> User Manual :</b> A document providing detailed instructions and guidance on how to use the app.
+ <b> Video Manual :</b> A video guide providing visual instructions on using the app's features.
+ <b> View Screen :</b> The screen used to view detailed information about a specific record.
+ <b> Zipping Completed : </b> The system has finished compressing the selected files or data into a ZIP archive, and they are now ready for transfer.
+ <b> Not applicable :</b> This category does not apply in this case.
+ <b> Not Yet Sanctioned :</b> The building has not received official approval or permission yet.
+ <b> Sanctioned House : </b > The houses or lands issued for the common people under Government Schemes.
+ <b> Sanctioned and Construction Not Started :</b> The building has been officially approved, but the construction work hasn't started yet.
+ <b> Sanctioned and Construction Stopped :</b> The building got official approval and construction had started, but the work has now been stopped before completion.
+ <b> Sanctioned and Construction Completed :</b> The building was officially approved, and the construction work is fully finished.
+ <b> Sanctioned and Under Construction :</b> The building has been officially approved, and the construction work is currently in progress.
+ <b>Upload Pending Records :</b> The number of records that are pending to upload.
+ <b>Uploaded Records :</b> The number of records that are successfully uploaded.
+ <b>Upload Failed Records :</b> The number of records that are failed to upload.
+ <b>Total Records :</b> The Total number of records (upload pending, uploaded, upload failed).
