# <u> User/Citizen Details</u>: Aadhaar Manual
Manual Entry (If QR/OCR Doesn't Work)

{{< notice "tip">}}
Sometimes, not all details can be captured automatically, so you need to enter them manually.
{{< /notice >}} <br>

If the Aadhaar Card fails to scan, you can enter the User Details Manually.
{{< notice "note">}}
You must fill all the Mandatory fields(<b><span style="color: Red;">*</span></b>) on the Registration Screen. 
{{< /notice >}} <br>
1.  Enter the <b>Aadhaar Number</b> of the User (Mandatory Field).
    >+  User Aadhaar Number should only start with numbers <b>2-9</b>.<br>
    >+  Allowed Characters <b>0-9</b><br>
    >+  Length <b>12</b><br>
1.  Enter the <b>Name</b> of the User (Mandatory Field).
    >+  User Name should only start with alphabets, but not with numbers and special characters.
    >+  Allowed Characters <b> - A-Z a-z 0-9</b><br>
    >+  Minimum Length <b>3</b><br>
    >+  Maximum Length <b>64</b><br>
1.  Enter the <b>Surname</b> of the User (Mandatory Field).
    >+  User Surname should only start with alphabets, but not with numbers and special characters.
    >+  Allowed Characters <b> _ - A-Z a-z 0-9</b><br>
    >+  Minimum Length <b>1</b><br>
    >+  Maximum Length <b>32</b><br>
1.  Enter the <b>Father/Spouse Name</b> of the User (Mandatory Field).
    >+  User Father/Spouse Name should only start with alphabets, but not with numbers and special characters.
    >+  Allowed Characters <b>_ - / : A-Z a-z 0-9</b><br>
    >+  Minimum Length <b>3</b><br>
    >+  Maximum Length <b>64</b><br>
1.  Select the <b>Gender</b> of the User (Mandatory Field).
1.  Select the <b>Date of Birth</b> of the User from the Calendar (Mandatory Field).
    {{< notice "note" >}} User's Age must be in between 15 - 45 years old.<br>{{< /notice >}}
1.  The <b>Age</b> of the User will be generated automatically based on the Date of Birth (Mandatory Field).
1.  Enter the <b>Address</b> of the User (Mandatory Field).
    >+  Allowed Characters <b>_ - / @ # & . : A-Z a-z 0-9</b><br>
    >+  Minimum Length <b>6</b><br>
    >+  Maximum Length <b>128</b><br>
