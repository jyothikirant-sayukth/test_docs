# <u>టెర్మినాలజీ</u>
+ <b> తిరిగిః</b> వినియోగదారుల ప్రస్తుత స్క్రీన్ను మునుపటి స్క్రీన్కు తరలించే చర్య.
+ <> బ్యాకప్ : </b> ఒక మొబైల్ నుండి మరొక మొబైల్కు డేటాను బదిలీ చేయడం.
+ <> పౌరుడు : </b> వారి స్థానిక పంచాయతీలో నివసించే వ్యక్తి.
+ <> పూర్తి సర్వే : </b> సర్వే పూర్తి అయిన ఒక గ్రామం/పంచాయతీని మూసివేయడానికి.
+ <> నిర్ధారణ స్క్రీన్ : </b> సృష్టించు స్క్రీన్లో ఇచ్చిన వివరాలను ధృవీకరించడానికి.
+ <> సృష్టించుకోండి సృష్టించండి స్క్రీన్ : </b> ఇన్పుట్లను ఇవ్వడానికి మరియు క్రొత్త రికార్డులను సృష్టించడానికి (ఉదా. కొత్త ఇంటిని సృష్టించడం).
+ <> డాష్బోర్డ్ : </b> యాప్ ఫీచర్ల సంక్షిప్త వివరణ లేదా సారాంశాన్ని ప్రదర్శించే హోమ్ స్క్రీన్.
+ <> ను తొలగించండి : </b> జాబితా తెర నుండి రికార్డును తొలగించడానికి.
+ <> పరికరం అనుకూలత : </b> ఇది పరికరం మద్దతిస్తుందో లేదో నిర్ధారించడానికి మొత్తం పరికరం అనుకూలత స్థితిని తనిఖీ చేస్తుంది.
+ <b> పూర్తయిందిః</b> ఇది పూర్తయినప్పుడు పనినిష్క్రమించడానికి చర్య.
+ <b> సవరించుః</b> రికార్డుల్లో ఉన్న సమాచారాన్ని సవరించడం లేదా నవీకరించడం.
+ <> (తరచుగా అడిగే ప్రశ్నలు) : </b> అనువర్తనం గురించి సాధారణ ప్రశ్నలకు సమాధానం ఇచ్చే విభాగం.
+ <> ను పూర్తి చేయండి : </b> రికార్డును పూర్తి చేసి, జాబితా స్క్రీన్లో సేవ్ చేసే చర్య.
+ <b> పాస్వర్డ్ మర్చిపోయానుః</b> వినియోగదారులు పాస్వర్డ్ను మరచిపోయినట్లయితే దాన్ని రీసెట్ చేయడానికి ఒక ఫీచర్.
+ <> మ్యాప్ : </b> భౌగోళిక డేటా లేదా స్థానాల దృశ్య ప్రాతినిధ్యం.
+ <b> సహాయంః</b> వినియోగదారులకు మార్గనిర్దేశం చేయడానికి సహాయం మరియు సూచనలను అందించే విభాగం.
+ <b> సమాచార చిహ్నంః</b> ఇది ఫీచర్ గురించి అదనపు సమాచారాన్ని అందిస్తుంది.
+ <b> జాబితా తెరః</b> సృష్టించిన రికార్డుల జాబితాను ప్రదర్శించే తెర.
+ <> లాగిన్ : </b> అనువర్తనానికి ప్రాప్యత పొందడానికి మీ ఆధారాలను (యూజర్పేరు మరియు పాస్వర్డ్) నమోదు చేసే ప్రక్రియ.
+ <> లోగోఅవుట్ : </b> అనువర్తనం నుండి నిష్క్రమణ చర్య.
+ <b> మెనుః</b> అనువర్తనం యొక్క వివిధ లక్షణాలకు ప్రాప్యత చేయడానికి ప్రధానావిగేషన్ ప్రాంతం.
+ <b> తదుపరిః</b> కరెంట్ స్క్రీన్ను తదుపరి స్క్రీన్కు కదిలించే వినియోగదారుల చర్య.
+ <b> నోటిఫికేషన్లుః</b> అప్లికేషన్కు పంపిన హెచ్చరికలు అప్డేట్స్ లేదా అనువర్తనానికి సంబంధించిన ఏదైనా సమాచారం గురించి వినియోగదారులకు తెలియజేయడానికి.
+ <> (ఆప్టికల్ క్యారెక్టర్ రికగ్నిషన్) : </b> ఇది ఆధార్ కార్డ్ యొక్క ఫోటోను సంగ్రహిస్తుంది మరియు సంగ్రహించిన ఫోటో యొక్క వివరాలను తెరపై ప్రదర్శిస్తుంది.
+ <> ప్రొఫైల్ : </b> వినియోగదారులు తమ వ్యక్తిగత వివరాలను చూడగలిగే విభాగం.
+ <b> పిన్ కరెంట్ స్థానంః</b> పిన్ ప్రస్తుత స్థానాన్ని పిన్ చేయడానికి ప్రస్తుత స్థానం సమీపంలో గుర్తించబడుతుంది.
+ <b> పిన్ ల్యాండ్మార్క్ స్థానంః</b> పిన్ ల్యాండ్మార్క్ స్థానం మీద గుర్తించబడుతుంది.
+ <b> పిన్ మునుపటి స్థానంః</b> పిన్ మునుపటి రికార్డు పిన్ స్థానంలో గుర్తించబడుతుంది.
+ <b> రికార్డుః</b> రికార్డు అనేది ఒక సర్వే తర్వాత సృష్టించబడిన డిజిటల్ ఎంట్రీ, ఇది ఒక ఆస్తి మరియు దానివాసితుల గురించి వివరణాత్మక సమాచారాన్ని సంగ్రహిస్తుంది.
+ <> నమోదు : </b> అనువర్తనాన్ని తెరవడానికి మరియు ఉపయోగించడానికి అవసరమైన వివరాలను అందించడం ద్వారా వినియోగదారు ఖాతాను సృష్టించే ప్రక్రియ.
+ <> శోధన బార్ : </b> జాబితా స్క్రీన్ ఎగువ భాగంలో ఉంది, ఇది నిర్దిష్ట రికార్డులను కనుగొనడానికి మిమ్మల్ని అనుమతిస్తుంది.
+ <b> సెట్టింగులుః</b> అప్లికేషన్ ఇంటర్ఫేస్ కోసం వినియోగదారులు తమ ప్రాధాన్యత భాషను ఎంచుకోవడానికి అనుమతించే సెట్టింగ్.
+ <b> గణాంకాలుః</b> ఇది జాబితా తెరపై రికార్డుల సంఖ్యను ప్రదర్శిస్తుంది.
+ <b> సర్వేః</b> నిర్దిష్ట ఆస్తి గురించి డేటాను సేకరించే ప్రక్రియ.
+ <> నిబంధనలు & షరతులు : </b> వినియోగదారు మరియు అనువర్తనం మధ్య చట్టపరమైన ఒప్పందం, నియమాలు మరియు బాధ్యతలను తెలియజేస్తుంది.
+ <> టైటిల్ : </b> ప్రతి స్క్రీన్ ఎగువ భాగంలో కనిపించే శీర్షిక.
+ <> ట్రబుల్షూటింగ్ : </b> ట్రబుల్షూటింగ్ అనేది ఒక పరికరం లేదా ఏదైనా వ్యవస్థతో సమస్యలను కనుగొనడం మరియు పరిష్కరించడం.
+ <> అప్లోడ్ : </b> సర్వే యాప్ నుండి పంచాయతీ సేవా క్లౌడ్ సర్వర్కు డేటాను పంపే చర్య.
+ <b> వాడుకరిః</b> వినియోగదారు అనేది ఒక ఉత్పత్తి లేదా అనువర్తనాన్ని ఉపయోగించే వ్యక్తి.
+ <> యూజర్ మాన్యువల్ : </b> అనువర్తనాన్ని ఎలా ఉపయోగించాలో వివరణాత్మక సూచనలు మరియు మార్గదర్శకాలను అందించే పత్రం.
+ <> వీడియో మాన్యువల్ : </b> అనువర్తనం యొక్క లక్షణాలను ఉపయోగించడానికి దృశ్య సూచనలను అందించే వీడియో గైడ్.
+ <> వీక్షణ స్క్రీన్ : </b> ఒక నిర్దిష్ట రికార్డు గురించి వివరణాత్మక సమాచారాన్ని వీక్షించడానికి ఉపయోగించే స్క్రీన్.
+ <> జిప్పింగ్ పూర్తయింది : </b> సిస్టమ్ ఎంచుకున్న ఫైళ్ళను లేదా డేటాను ZIP ఆర్కైవ్లోకి కుదించడం పూర్తయింది, మరియు అవి ఇప్పుడు బదిలీకి సిద్ధంగా ఉన్నాయి.
+ <b> వర్తించదుః</b> ఈ వర్గం ఈ సందర్భంలో వర్తించదు.
+ <b> ఇంకా ఆమోదించబడలేదుః</b> ఈ భవనం ఇంకా అధికారిక ఆమోదం లేదా అనుమతి పొందలేదు.
+ <> పబ్లిక్ హౌస్ : </b > ప్రభుత్వ పథకాల కింద సామాన్య ప్రజలకు జారీ చేయబడిన ఇళ్ళు లేదా భూములు.
+ <b> ఆమోదం పొందింది మరియు నిర్మాణం ప్రారంభం కాలేదుః</b> భవనం అధికారికంగా ఆమోదించబడింది, కానీ నిర్మాణ పనులు ఇంకా ప్రారంభించబడలేదు.
+ <b> నిషేధించబడింది మరియు నిర్మాణం నిలిపివేయబడిందిః</b> భవనం అధికారిక ఆమోదం పొందింది మరియు నిర్మాణం ప్రారంభమైంది, కానీ ఇప్పుడు పని పూర్తయ్యే ముందు నిలిపివేయబడింది.
+ <b> ఆమోదించబడింది మరియు నిర్మాణం పూర్తయిందిః</b> భవనం అధికారికంగా ఆమోదించబడింది, మరియు నిర్మాణ పనులు పూర్తిగా పూర్తయ్యాయి.
+ <b> ఆమోదించబడిన మరియు నిర్మాణంలో ఉందిః</b> భవనం అధికారికంగా ఆమోదించబడింది, మరియు నిర్మాణ పనులు ప్రస్తుతం జరుగుతున్నాయి.
+ <>అప్లోడ్ పెండింగ్ రికార్డ్స్ : </b> అప్లోడ్ చేయబడే రికార్డుల సంఖ్య.
+ <> అప్లోడ్ చేయబడింది రికార్డులు : </b> విజయవంతంగా అప్లోడ్ చేయబడిన రికార్డుల సంఖ్య.
+ <> అప్లోడ్ విఫలమైంది రికార్డులు : </b> అప్లోడ్ చేయడంలో విఫలమైన రికార్డుల సంఖ్య.
+ <>మొత్తం రికార్డులు : </b> రికార్డుల మొత్తం సంఖ్య (అప్లోడ్ పెండింగ్, అప్లోడ్, అప్లోడ్ విఫలమైంది).
